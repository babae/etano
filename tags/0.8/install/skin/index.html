This wizard will guide you through the install process.

Step 1 - checking environment

  • {output.phpversion} PHP version higher than 4.4.0

    Although Etano may run on older versions of PHP we consider this to be a security risk.
    PHP 4.4.0 was released on 11-Jul-2005. If your host hasn't been able to update in such a long time you should move to another host.

  • {output.mysql} Mysql support

    Etano needs a MySql database to store all member information.

  • {output.sapi} PHP Server API

    If your PHP is configured as a CGI plug-in rather than an Apache module you may experience problems, as this configuration is not always working.

  • {output.write} Write support

    In order for this installer to make changes to files and for the application to write the cache files, write support is needed.

  • {rw_files.check_result} '{rw_files.file}' {rw_files.perms} permissions

    {rw_files.type} {rw_files.file} must have {rw_files.perms} permissions.

  • {output.tmp_perm} 'tmp' folder 777 permissions

    The tmp folder must have 777 permissions.

  • {output.tmpadmin_perm} 'tmp/admin' folder rwx permissions

    The 'tmp/admin' folder must have 777 permissions
