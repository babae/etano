All fields below are mandatory and must not be left empty.

Step 2 - site configuration

Site Location & Name
The name you put here will appear in the site header, in the title tag and in all pages where there are references to this site.
This is the URL where your site will be installed. Usually autodetected. Please note that http://myserver.com is not the same with http://www.myserver.com
The disk path on the server to where your site is installed. Usually autodetected.
Database Settings
This is the host where your mysql database is installed. Usually 'localhost' but could be any IP or server name. You must have received details about the database in the welcome email from your host.
Test DB connection. Once you fill in the database server, user, password and database name you can check if the application can connect using the supplied info.
This database must already exist. You must create a database in your server control panel prior to installing Etano.
All Etano-related tables in the database will be prefixed with this string.
Connection Settings
In order for the application to write files to disk, it needs to connect by ftp. You must have received details about the ftp connection in the welcome email from your host. Please leave the ftp host as 'localhost' and only if it fails the check below enter the server IP address.
Test FTP connection. Once you fill in the ftp server, user, password and path you can check if the application can connect using the supplied info.
This is the equivalent of the Base Path above but when you connect by ftp. Usually this is the ending part of the Base Path. Autodetected on Unix systems. Must end in a forward slash (/). At the very least this field should contain a forward slash.
